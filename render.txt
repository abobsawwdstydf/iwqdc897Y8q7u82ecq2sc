# 🚀 БЫСТРЫЙ ДЕПЛОЙ НА RENDER.COM

## ⚡ Вариант 1: Вместе (рекомендуется)

### 1. Создайте репозиторий на GitHub
```bash
git init
git add .
git commit -m "Deploy"
git push -u origin main
```

### 2. Создайте файл `.render.yaml` в корне:
```yaml
services:
  - type: web
    name: nimbus-server
    env: node
    region: frankfurt
    plan: free
    branch: main
    rootDir: apps/server
    buildCommand: npm install && npm run build
    startCommand: node dist/index.js
    envVars:
      - key: NODE_ENV
        value: production
      - key: JWT_SECRET
        generateValue: true
      - key: DATABASE_URL
        fromDatabase:
          name: nimbus-db
          property: connectionString
      - key: CORS_ORIGINS
        value: https://nimbus-web.onrender.com
      - key: ENCRYPTION_KEY
        generateValue: true

  - type: web
    name: nimbus-web
    env: static
    region: frankfurt
    plan: free
    branch: main
    rootDir: apps/web
    buildCommand: npm install && npm run build
    staticPublishPath: ./dist
    envVars:
      - key: VITE_API_URL
        value: https://nimbus-server.onrender.com

databases:
  - name: nimbus-db
    databaseName: nimbus
    user: nimbus
    plan: free
    region: frankfurt
```

### 3. Задеплойте на Render:
1. https://render.com → Login with GitHub
2. New + → Blueprint
3. Выберите репозиторий
4. Apply

### 4. Инициализируйте БД:
```bash
# В Shell сервера на Render:
cd apps/server
npx prisma migrate deploy
npx prisma db seed
```

### 5. Готово!
- Сервер: https://nimbus-server.onrender.com
- Клиент: https://nimbus-web.onrender.com

## ⚡ Вариант 2: Раздельно

### СЕРВЕР:

1. New + → Web Service
2. Настройки:
   - Name: `nimbus-server`
   - Root Directory: `apps/server`
   - Build: `npm install && npm run build`
   - Start: `node dist/index.js`

3. Переменные:
```
NODE_ENV=production
JWT_SECRET=<64 символа>
DATABASE_URL=<от PostgreSQL>
CORS_ORIGINS=<ваш фронтенд>
ENCRYPTION_KEY=<64 символа>
```

4. Создайте PostgreSQL:
   - New + → PostgreSQL
   - Name: `nimbus-db`
   - Скопируйте connectionString

5. Инициализация:
```bash
cd apps/server
npx prisma migrate deploy
npx prisma db seed
```

### КЛИЕНТ:

1. New + → Static Site
2. Настройки:
   - Name: `nimbus-web`
   - Root Directory: `apps/web`
   - Build: `npm install && npm run build`
   - Publish: `dist`

3. Переменные:
```
VITE_API_URL=https://nimbus-server.onrender.com
```

## 🔧 Команды для продакшена:

```bash
# Сборка
npm run build

# Запуск сервера

# Миграции
npm run db:push -w apps/server

# Сиды
npm run db:seed -w apps/server
```
