# 🚀 БЫСТРЫЙ СТАРТ - Nexo Messenger

## 1️⃣ Установка (5 минут)

```bash
# 1. Установи зависимости
npm install

# 2. Настрой базу данных
npm run db:setup

# 3. Запусти Redis (если нет Docker)
docker run -d -p 6379:6379 redis:latest

# 4. Запусти проект
npm run dev
```

**Всё!** Мессенджер доступен на `http://localhost:5173`

---

## 2️⃣ Настройка для Render (10 минут)

### Шаг 1: База данных
1. Создай PostgreSQL на Render: https://render.com
2. Скопируй **External Database URL**
3. Вставь в `.env` → `DATABASE_URL=...`

### Шаг 2: Telegram боты
1. @BotFather → `/newbot`
2. Создай канал, добавь бота админом
3. @RawDataBot → узнай ID канала
4. Вставь в `.env`:
   ```
   TELEGRAM_BOT_TOKENS=токен
   TELEGRAM_CHANNEL_IDS=-1001234567890
   ```

### Шаг 3: Discord (опционально)
1. Настройки канала → Интеграции → Вебхуки
2. Скопируй URL
3. Вставь в `.env`: `DISCORD_WEBHOOK_URLS=...`

### Шаг 4: Деплой
1. Render → New Web Service
2. Выбери репозиторий
3. Build: `npm install && npm run build`
4. Start: `npm run dev`
5. Добавь все переменные из `.env` в Environment Variables

## 🆘 Проблемы?

### "No resources available"
- Проверь токены ботов в .env
- Убедись, что боты - админы каналов

### Ошибка Redis
```bash
docker run -d -p 6379:6379 redis:latest
```

### Ошибка БД
- Проверь DATABASE_URL
- Запусти `npm run db:setup`
