## 🆘 Проблемы?

### "No resources available"
- Боты уже добавлены в .env
- Каналы уже добавлены в .env
- Просто запусти `npm run go`

### Ошибка Redis
```bash
docker run -d -p 6379:6379 redis:latest
```

Или в .env добавь:
```
REDIS_URL=redis://localhost:6379
```
